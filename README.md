# PowerLevel.nvim

# NOT FUNCTIONAL YET

A Neovim plugin for Markdown todo lists in which you may go super saiyan.

## Installation

### Using `packer.nvim`:

```lua
use {
    'NeoManslayer/powerlevel.nvim',
    config = function()
        require("powerlevel").setup({
            -- options
        })
    end
}
```

### Using `vim-plug`:

```lua
Plug 'NeoManslayer/powerlevel.nvim'
```

### Using `lazy.nvim`:

```lua
{ 'NeoManslayer/powerlevel.nvim' }
```
